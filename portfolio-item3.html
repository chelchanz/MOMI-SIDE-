Menentukan Bilangan Terbesar dari 3 bilangan

Kalimat Deskriptif

1. Start

2. Tipe data yang digunakan adalah integer bilangan bulat

3. Pengguna memasukkan bilangan 1, bilangan 2, bilangan 3

4. Jika bilangan 1 lebih besar dari bilangan 2 dan bilangan 1 lebih besar dari bilangan 3, maka bilangan terbesar adalah bilangan 1

5. Jika bilangan 2 lebih besar dari bilangan 1 dan bilangan 2 lebih besar dari bilangan 3, maka bilangan terbesar adalah bilangan 2

6. Jika tidak, maka bilangan 3 adalah bilangan terbesar

7. Setelah itu, program akan menampilkan bilangan terbesar menggunakan fungsi cout ke layar

8. Selesai

Flowchart

Coding

#include <iostream>
using namespace std;

int main() {
    int bilangan_terbesar, bil_1, bil_2, bil_3;
    
    cout << "Masukkan bilangan pertama: ";
    cin >> bil_1;
    cout << "Masukkan bilangan kedua: ";
    cin >> bil_2;
    cout << "Masukkan bilangan ketiga: ";
    cin >> bil_3;
    
    if (bil_1>bil_2 && bil_1>bil_3) {
    	bilangan_terbesar= bil_1;
    } else if (bil_2>bil_1 && bil_2>bil_3) {
	    bilangan_terbesar = bil_2;
    } else {
    	bilangan_terbesar = bil_3;
	}
    
    cout << "Bilangan terbesar adalah: " << bilangan_terbesar << endl;
    return 0;
}
