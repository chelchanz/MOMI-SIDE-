Menentukan Bilangan Ganjil dan Genap

Kalimat Deskriptif

1. Start

2. Tipe data yang digunakan adalah integer karena input yang digunakan adalah bilangan bulat

3. Program meminta pengguna memasukkan bilangan bulat

4. Bilangan tersebut diperiksa menggunakan operator modulus (%)

5. Jika bilangan tersebut habis dibagi 2, maka bilangan tersebut adalah genap

6. Jika sisa pembagian tidak sama dengan 0, maka bilangan tersebut adalah ganjil

7. Selesai

Flowchart

Coding

#include <iostream>
using namespace std;

int main() {
    int x;
    
    cout <<"Masukkan bilangan bulat: ";
    cin >> x;
    
    if (x % 2 == 0) {
    	cout << x << " adalah bilangan genap";
    }
    else {
    	cout << x << " adalah bilangan ganjil";
    }
    cout << endl;
    return 0;
}
